Project 2: Basic Webscraper

This is was my dive into python, I used a mix of Beautiful Soup 4 and Selenuite to scrape a webpage for information and save it in a miniature database. I then parse out the relevant information so that when the user tries to look up a specific champion counter, the program responds with the top 3 results. This project also emplyed the use of GeckoEngine to launch an instance of firefox as the web page I was trying to scrape used a lot of JavaScript that needed to be loaded before it could be scraped.
